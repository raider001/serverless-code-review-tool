# Managing Reviews
This guide explains how to join, leave, and provide approval for reviews.
## Joining a Review
1. Open the review
2. Click **Join Review** button (top right)
3. Your name appears in the reviewers list
4. All buttons become enabled
After joining you can:
- Add and edit comments
- Mark comments as resolved
- Approve or request changes
- Edit review details
## Leaving a Review
1. Click **Leave Review** button
2. Your name is removed from reviewers
3. Cannot add new comments or approve
## Providing Feedback
### Approve Changes
Click **Approve** when code is ready to merge.
### Request Changes
Click **Request Changes** when modifications are needed.
## Review States
- **OPEN**: Initial state
- **IN PROGRESS**: Active reviewing
- **APPROVED**: All reviewers approved
- **CHANGES REQUESTED**: Modifications needed
##Permissions
**Non-Reviewers**:
- View files and diffs
- Read comments
- Cannot interact
**Reviewers**:
- Everything above plus
- Add comments
- Approve/request changes
- Edit review
