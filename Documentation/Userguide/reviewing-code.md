# Reviewing Code
This guide explains how to navigate, view diffs, and add comments during a code review.
## Opening a Review
1. Navigate to the **Review Selection** tab
2. Select a review from the list
3. Click **Load Review** or double-click
## Review Panel Overview
### Review Header
- Status badge, title, Edit/Join/Leave buttons
- Author, description, and reviewers list
### File Navigation
Left sidebar with all changed files:
- Green (A): Added files
- Yellow (M): Modified files  
- Red (D): Deleted files
- Blue (R): Renamed
### Diff Viewer
Shows code changes with syntax highlighting:
- Green background: Added lines
- Red background: Removed lines
- White: Context
### Comment Indicators
- Blue circle: Observation
- Orange circle: Unresolved issue
- Green circle: Resolved
## Adding Comments
1. Double-click a line number
2. Enter comment text
3. Check "Mark as Needs Resolution" if it's an issue
4. Click Save
## Viewing Comments
Click the colored circle to see comment threads.
